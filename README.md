# IT Infrastructure & Asset Management Portfolio

Welcome to my portfolio repository! This repository contains comprehensive projects and analyses related to network topology design, IT asset management (GLPI), incident management, and visual documentation.

## 📋 Portfolio Components

### 1. Network Topology Projects
- **Router-on-a-Stick Architecture**: Terraform configurations for a simple network topology with logical segmentation (VLANs)
- **Asset Management Analysis**: Deep dive into how GLPI can be utilized within each topology
- **Incident Scenarios**: Documentation of potential network incidents and their impacts

### 2. Documentation
- **Vulnerability Assessment**: Identification and analysis of security weaknesses
- **Risk Mitigation Strategies**: Solutions to reduce incident likelihood
- **Configuration Best Practices**: Real-world implementations and security hardening
- **International Standards Compliance**: ISO 27001, NIST, and other frameworks

### 3. Visual Documentation
- **Packet Tracer Topologies**: Screenshots and detailed diagrams
- **Figma Architecture Diagrams**: Comprehensive visual representations including:
  - User roles and RBAC (Role-Based Access Control)
  - GDPR compliance considerations
  - Physical and logical assets
  - Monitoring tools integration (Zabbix, GLPI)
  - Color-coded relationships and dependencies
